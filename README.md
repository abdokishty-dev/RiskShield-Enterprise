# 🛡️ RiskShield-Enterprise (ERM Pro v3.0)

**RiskShield-Enterprise** هو منظومة ذكية متكاملة ومتقدمة لإدارة المخاطر المؤسسية (ERM)، تم تصميمها وتطويرها لتربط بين ذكاء الآلة وآليات الرقابة المالية الصارمة، متوافقة مع أعلى معايير التدقيق والتقارير المالية (**FAR & AUD**).

## 🚀 الميزات الرئيسية (Key Features)

* 🔍 **التدقيق والرقابة المستمرة (Continuous Auditing - AUD):** نظام دفاعي ذكي يعمل كخط رقابي أول لرصد الانحرافات، الأخطاء، ومؤشرات الاحتيال قبل حدوثها.
* 📊 **ربط التقارير المالية (Financial Reporting - FAR):** تحليل وربط أثر المخاطر مباشرة بالقوائم المالية والمخصصات لضمان أعلى درجات الموثوقية والدقة.
* 🤖 **مستشار ذكي (AI-Powered Insights):** محرك تنبؤي مدعوم بالذكاء الاصطناعي لتقديم تحليلات استباقية وتوقعات مالية دقيقة.
* 🎯 **مؤشرات المخاطر (KRI & Heatmaps):** لوحة متابعة تفاعلية (Dashboard) تشمل خريطة حرارية وتوزيعاً دقيقاً لدرجات الخطورة والفئات.
* 💾 **مرونة البيانات:** يعتمد على قاعدة بيانات `SQLite` مدمجة مع دعم كامل لتصدير واستيراد البيانات عبر `Excel`.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Backend & Logic:** Python / Node.js (حسب التقنية الحالية للـ Backend)
* **Database:** SQLite
* **Frontend:** Clean HTML5, CSS3, & JavaScript
* عبد الفتاح قشطي
* مطور نظم ورقابة مالية
* المنظومة الحديثة للإدارة المالية
* صدقة جارية على روح والدي الحبيب
